Welcome to Scylla

Scylla is a tool designed to help you monitor and track changes to URLs. There are two ways we help you manage change:

Single URL Reports

With Single URL Reports, we take a screenshot of a specific URL and allow you to compare the current state with a known master (i.e. what you expect it to look like).

These reports must be collected into a Batch of reports, which can be launched by clicking on the Rocket when viewing the Batch Details. You may also choose to schedule a batch to run at a specific time. After a batch is run on a schedule, it will email the watchers with a short summary of the results, a great way to be greeted every morning when you come in to work.

AB Comparisons

With AB Compare, we take a screenshot of two separate URLs and compare them to each other. This tends to be a great tool to use while in active development or refactoring.

Visual Differences

Our Visual Diff tool is available to inspect differences, regardless of which path you choose. The tool has two modes to help you identify and analyze differences:

Swipe

In this mode, the slider bar will let you swipe between the Master / Current or Url A / Url B. By sliding, you will be able to see a seam where the image changes, helping you identify exactly what the differences are. You'll be able to count the pixels an element moved, or see two colors side-by-side.

Diff Overlay

In this mode, the slider bar will adjust the visibility of a map of changed pixels. This map shows each and every pixel that changed in bright red, with the rest of the image muted. By reducing the transparency, you will be able to see the original image more strongly.
